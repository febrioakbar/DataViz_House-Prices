About Dataset

Downloaded from Kaggle : https://www.kaggle.com/c/house-prices-advanced-regression-techniques

The Ames Housing dataset was compiled by Dean De Cock for use in data science education. Its an incredible alternative for data scientists looking for a modernized and expanded version of the often cited Boston Housing dataset.

Ask a home buyer to describe their dream house, and they probably wont begin with the height of the basement ceiling or the proximity to an east-west railroad. But this playground competitions dataset proves that much more influences price negotiations than the number of bedrooms or a white-picket fence.

With 79 explanatory variables describing (almost) every aspect of residential homes in Ames, Iowa, this competition challenges you to predict the final price of each home.

1. Count of one categorical feature

Lot Configuration is a variable that state type of configuration of each house.

df %>% 
  group_by(LotConfig) %>% 
  summarise(nbrow=n()) %>% 
  ggplot(aes(x = LotConfig,y=nbrow))+
  coord_cartesian(ylim=c(0,1100))+
  geom_bar(stat='identity',fill = "#2B76CC")+
  geom_label(aes(label=nbrow))+
  labs(title =  'Frequence of Lot Configuration Category',subtitle = 'Sales or marketer must know the trend of lot configuration over time', x ='Lot Configuration', y='Count')+theme_bw()

The bar chart show us that the most used lot configuration type by far is inside type and the rarest is FR3 with just 4 data.

3. Categorical-continuous

Categorical = Sale Condition Continous = Sale Price

df %>%
  ggplot(aes(x = SaleCondition, y = SalePrice, color=SaleCondition)) +
  geom_boxplot(notch=FALSE,outlier.colour="red",outlier.size=2)+
  labs(title = 'Categorize Saleprice by Sale Condition',
  subtitle = 'Sales can find out witch sale condition is the highest and the anomalies')+
  theme(axis.text.x = element_text(angle = 90, vjust = 0.5, hjust=1))+
  facet_grid(. ~ CentralAir)+theme_bw()

From the boxplot we know that partial sale condition is the overall highest but the normal condition have outlier more than the others

4. Continuous - Continuous

df %>%
  ggplot(aes(x = YearBuilt, y = SalePrice, col='red')) +
  geom_point(alpha=0.5, size=2) +
  geom_smooth(formula = y ~ x,method='lm',  linetype="dashed",
             color="darkred", fill="blue")+
  labs(title = 'YearBuilt and Saleprice correlation',
  subtitle = 'Sales can find out if yearbuilt of a house can affect its price')+
   facet_wrap(~YrSold)

From the scatterplot we know that yearbuilt is affecting the saleprice with positive correlation, so the oldest the house than the price will be cheaper.
